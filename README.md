# Canvas. — Visual Checklist Builder

A modern, fully client‑side visual workflow and checklist builder.  
Create interactive troubleshooting flows, branching questions, loops, and more – all with a premium dark UI.

## Features

- **Visual node editor** with drag‑and‑drop
- Node types: Choice, Decision, Message (Info/Success/Warning/Error/Action), Copy Box, Input, Number, End
- Branching paths and loops
- Smooth SVG connection lines
- Zoom, pan, minimap, and marquee selection
- Multiple tabs / projects
- Auto‑save to localStorage
- Export as `.canvas` (XML inside)
- Import `.canvas` or paste XML
- Interactive runtime mode with variable support
- Undo/Redo, keyboard shortcuts, and recent files
- HSBC‑red themed UI

## Running locally

Just open `index.html` in any modern browser.  
No build tools or external dependencies required.
